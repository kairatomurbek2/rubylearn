Задание 6-1 (телефонная книга)
создание массива контакты с включенными в него хэшами в качестве элементов
    создание в каждом из хэшей ключа - имя:,значения - "имя человека" и ключа телефон:, значения - "номер телефона"

создание метода поиск_по_имени с включенными в него переменными (контакты_из_массива, искомое_пользователем_имя)
    для каждого из элементов контакты_из_массива сделать следующее:
        если ключ элемента == искомое_пользователем_имя
            возвратить значение данного элемента
        конец условия
    конец действия
    возвратить ничего
конец метода

создание метода вводный_знак()
    печать без переноса строки сообщения ">" с пробелом
    возвратить ответ пользователя введенный на экране
конец метода

    начало бесконечного цикла

        вывод на экран сообщения о том, какую команду хочет ввести пользователь?
        введенная_команда = метод вводный_знак()

        начало кейса введенная_команда
        в случае, когда введенная_команда равняется "list"
            с каждым из элементов массива контакты произвести следующие действия
                вывести на экран сообщение с именем из хэша
                вывести на экран сообщение с телефоном из хэша
            конец действия

        в случае, когда введенная_команда равняется "find"
            вывод на экран сообщения о том, чьи именно контакты хочет найти пользователь
            искомое_пользователем_имя = метод вводный_знак()

            искомый_контакт = метод поиск_по_имени(контакты_из_массива, искомое_пользователем_имя)

            если искомый_контакт равняется ничему
                вывод на экран сообщения, что такой контакт не найден
            иначе
                вывести на экран сообщение с именем из хэша в массиве контакты
                вывести на экран сообщение с телефоном из хэша в массиве контакты
            конец условия

                вывод на экран сообщения, о необходимости ввода номера телефона

                добавить в значение хэша_с_добавленными_контактам введенное_имя
                добавить в значение хэша_с_добавленными_контактам введенный_номер

                добавить в массив контакты хэш_с_добавленными_контактам

                вывод на экран сообщения об успешном добавлении контакта

        иначе
