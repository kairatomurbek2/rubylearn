Задание 6-2 (рейтинг фильмов)
создание хэша база_фильмов, в котором в качестве ключа выступает название фильма,
в качестве значения - другой хэш с ключом имя оценивающего и значением рейтинг оценивающего

создание метода вводный_знак()
    печать без переноса строки сообщения ">" с пробелом
    возвратить ответ пользователя введенный на экране
конец метода

создание метода сравнение_фильмов_по_ключу с включенными в него переменными(база_фильмов, название_фильма)
    для каждого из элементов хэша база_фильмов |ключ, значение|сделать следующее:
        если ключ == название_фильма
            возвратить ключ
        конец условия
    конец действия
    возвратить ничего
конец метода

создание метода сравнение_фильмов_по_значению с включенными в него переменными(база_фильмов, название_фильма)
    для каждого из элементов хэша база_фильмов |ключ, значение|сделать следующее:
        если ключ == название_фильма
            возвратить значение
        конец условия
    конец действия
    возвратить ничего
конец метода

создание метода вычисление_среднего_значения с включенными в него переменными(рейтинги)
    результат = 0
    для каждого из рейтинг |ключ, значение|сделать следующее:
        результат = результат + значение
    конец действия

    возвратить результат / количество рейтингов с дробным окончанием
конец метода

начало бесконечного цикла

    вывод на экран сообщения о том, какую команду хочет ввести пользователь
    введенная_команда = метод вводный_знак()

    начало кейса введенная_команда

    в случае, когда введенная_команда равняется "list"
        с каждым из элементов хэша база_фильмов произвести следующие действия
            вывести на экран сообщение с ключом и значением
        конец действия

    в случае, когда введенная_команда равняется "add"
        создать хэш новый_фильм = {}

        вывод на экран сообщения о необходимости ввода названия фильма
        введенное_название = метод вводный_знак()

        сравнение_названий = метод сравнение_фильмов_по_ключу(база_фильмов, введенное_название)

        если сравнение_названий равняется ничему
            добавить введенное_название в качестве ключа в хэш новый_фильм с пустым значением
            добавить хэш новый_фильм в хэш база_фильмов
            вывод на экран сообщения об успешном добавлении фильма
        иначе
            вывод на экран сообщения о том, что такой фильм уже существует
        конец условия

    в случае, когда введенная_команда равняется "add rating"
        создать хэш новый_рейтинг = {}

        вывод на экран сообщения о необходимости ввода названия фильма
        введенное_название = метод вводный_знак()

        сравнение_названий = метод сравнение_фильмов_по_значению(база_фильмов, введенное_название)

        если сравнение_названий равняется ничему
            вывод на экран сообщения о том, что такой фильм уже существует
        иначе
            вывод на экран сообщения о необходимсоти ввода имени пользователя
            имя = метод вводный_знак()
            вывод на экран сообщения о необходимсоти ввода рейтинга для выбранного фильма
            рейтинг = метод вводный_знак() переведенный в целое число

                если рейтинг меньше 0 и больше 10
                    вывод на экран сообщения о том, что нужно ввести число с 1 до 10
                иначе
                    добавить в хэш новый_рейтинг имя(ключ) и рейтинг (значение) пользователя
                    добавить хэш новый_рейтинг в сравнение название - возвращенное методом сравнение_фильмов_по_значению значение
                    вывод на экран сообщения об успешном добавлении рейтинга для фильма
                конец условия
        конец условия

    в случае, когда введенная_команда равняется "see rating"
        вывод на экран сообщения о необходимости ввода названия фильма
        введенное_название = метод вводный_знак()

        сравнение_названий = метод сравнение_фильмов_по_значению(база_фильмов, введенное_название)

        если сравнение_названий равняется ничему
            вывод на экран сообщения о том, что такой фильм уже существует
        иначе
            вывод на экран сообщения о рейтинге введенного_названия
            с каждым из элементов сравнение_названий |ключ, значение| сделать следующее:
                вывод на экран сообщения о том, что ключ имеет такое-то значение
            конец действия
        конец условия

        средняя = метод вычисление_среднего_значения(сравнение_названий)
        вывод на экран сообщения о среднем рейтинге для введенного_названия фильма

    в случае, когда введенная_команда равняется "all ratings"
        с каждым из элементов базы_фильмов |ключ, значение| сделать следующее:
            средняя = метод вычисление_среднего_значения(значение)
            вывод на экран сообщения о осреднем рейтинге каждого из фильмов базы_фильмов
        конец действия

    в случае, когда введенная_команда равняется "exit"
        вывод на экран сообщения о выходе
        прерывание бесконечного цикла
    иначе
        вывод на экран сообщения о том, что введенная_команда пользователем не существует в программе
    конец условия

конец бесконечного цикла
